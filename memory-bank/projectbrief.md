# 프로젝트 개요 (Project Brief)

## 프로젝트명
managesys

## 핵심 목표
- 관리 시스템 웹 애플리케이션 (프론트: Vue 3, 백엔드: Spring Boot)
- 사용자 인증/권한, 고객 관리 등 관리 기능 제공

## 범위
- **프론트엔드**: Vue 3, Vue Router, Pinia, AG Grid, Axios
- **백엔드**: Spring Boot 4.x, Java 17, Spring Security, JWT, MyBatis, MariaDB
- **DB**: MariaDB (localhost:3307/managesys)

## 주요 기능(현재/예정)
- 로그인/인증 (JWT)
- 사용자 관리
- 고객 관리
- 공지 등 공통 UI 컴포넌트

## 제약·가정
- 백엔드 서버 포트: 8085
- 프론트 개발 서버: Vue CLI 기본 포트
